## 🐛 TROUBLESHOOTING

### "AI Connection Error" on startup
**Solution**: Check your internet connection. The tool needs internet to reach Puter's API.

### "AI timeout - please try again"
**Solution**: AI call took too long. Just run the investigation again - it usually works the second time.

### "AI API Error"
**Solution**: 
1. Check that you're online
2. Verify the API key hasn't changed
3. Try again in a few seconds

### Image analysis not working
**Solution**:
1. Make sure image is in a supported format (jpg, png, gif, bmp)
2. Image should be under 20MB
3. Image should clearly show a face
